**Synthesized Improvement Proposal for Surrogate CLI Help Command**

## Diagnosis
The Surrogate CLI faces several usability challenges that hinder user experience, particularly for newcomers. Key issues include:

1. **Lack of Comprehensive Help Command**: Users cannot access detailed information about available commands and options.
2. **Inconsistent Command Descriptions**: Some commands lack clear descriptions, leading to confusion about their functionality.
3. **No Examples for Usage**: Users do not have examples of how to use commands, which could help them understand the expected input and output.
4. **Limited Error Handling**: If a user inputs an incorrect command, there is no helpful feedback or guidance on how to correct it.
5. **No Version Information**: Users cannot check the current version of the CLI, which is important for troubleshooting and support.
6. **No Interactive Help**: The CLI lacks an interactive help feature that could guide users through commands and options.

## Proposed Change
Enhance the help command functionality in the Surrogate CLI by:

* Updating the `help` command implementation in `cli.py` (or the relevant file where command handling is defined).
* Adding detailed descriptions and examples for each command.
* Implementing version information display.
* Improving error handling for unrecognized commands.
* Introducing an interactive help feature to guide users through commands and options.

    Examples:
    - sg "fix the login bug" # One-shot task execution
    - surrogate plan set TODO.md # Set tasks from a markdown file
    Version: 1.0.0
    """
    print(help_text)

# Update command handling to include help command

* Add an interactive help feature that guides users through commands and options.

### Interactive Help Code Snippet
```python
# cli.py
def interactive_help():
    print("Welcome to the Surrogate CLI interactive help!")
    print("Type 'commands' to see a list of available commands.")
    print("Type 'options' to see a list of available options.")
    while True:
        user_input = input("> ")
        if user_input == "commands":
            print("Available commands:")
            print("- surrogate # Start interactive REPL")
            print("- surrogate 'task' # Execute one-shot task")
            print("- surrogate --auto # Plan-driven auto mode")
            # ...
        elif user_input == "options":
            print("Available options:")
            print("- -h, --help # Show this help message and exit")
            print("- --auto # Plan-driven auto mode")
            # ...
        else:
            print("Invalid input. Type 'commands' or 'options' for help.")
```
