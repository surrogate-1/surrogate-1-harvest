#### Implementation Plan

1. **Identify Public Dataset Files**: Use the Hugging Face API to list the public dataset files for a specific repository. This can be done using the `list_repo_tree` method with `recursive=False` to get the file paths for a single date folder.
2. **Save File List to JSON**: Save the list of file paths to a JSON file. This will be used to embed the file list in the training script.
3. **Embed File List in Training Script**: Modify the training script to use the CDN-only fetches with zero API calls during data load. This can be done by reading the JSON file containing the file list and using the `https://huggingface.co/datasets/{repo}/resolve/main/{path}` URL to download the files.
4. **Test the Implementation**: Test the implementation by running the training script and verifying that it can download dataset files from the CDN without hitting the rate limit.
