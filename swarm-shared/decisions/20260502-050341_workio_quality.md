### High-Value Incremental Improvement for Workio Quality
#### Diagnosis
The Workio project requires enhancements in its quality to improve the overall system's functionality, efficiency, and user experience. Based on the patterns and lessons learned, the highest-value incremental improvement that can ship in <2h is to implement a script to automate the execution of `knowledge-rag` after running market analysis scripts, such as `granite-business-research.sh`, to provide contextual insights.

#### Implementation Plan
1. **Create a new script**: Create a new Bash script, e.g., `run_knowledge_rag.sh`, in the `/opt/axentx/workio` directory.
2. **Add shebang and make executable**: Add the Bash shebang `#!/usr/bin/env bash` to the top of the script and make it executable using `chmod +x run_knowledge_rag.sh`.
3. **Invoke knowledge-rag**: Add a line to the script to invoke `knowledge-rag` with the required parameters, e.g., `knowledge-rag --query "top hub and related docs"`.
4. **Schedule the script**: Schedule the script to run after the market analysis script using a tool like `cron`.

#### Code Snippet
```bash
#!/usr/bin/env bash

# Invoke knowledge-rag
knowledge-rag --query "top hub and related docs"
```
#### Example Use Case
To use this script, simply run `bash run_knowledge_rag.sh` after executing the market analysis script. This will provide contextual insights by querying the top hub and related documents.
