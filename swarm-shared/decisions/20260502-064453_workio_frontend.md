### Implementation Plan
#### Step 1: Update `server/src/routes/line.ts` to Handle Webhook Requests
Create a new endpoint to handle LINE OA webhook requests. This will involve updating the Express.js route to parse the incoming request and trigger the necessary actions.

```typescript
// server/src/routes/line.ts
import express, { Request, Response } from 'express';
import { WebClient } from '@line/bot-sdk';

const lineRouter = express.Router();
const webClient = new WebClient(process.env.CHANNEL_ACCESS_TOKEN);

lineRouter.post('/webhook', (req: Request, res: Response) => {
  const events = req.body.events;
  events.forEach((event) => {
    if (event.type === 'message') {
      // Handle incoming message
      handleIncomingMessage(event);
    } else if (event.type === 'follow') {
      // Handle follow event
      handleFollowEvent(event);
    }
  });
  res.status(200).send('Webhook received');
});

export default lineRouter;
```

#### Step 2: Implement `handleIncomingMessage` and `handleFollowEvent` Functions
Create functions to handle incoming messages and follow events. These functions will trigger the necessary actions, such as sending notifications or updating user data.

```typescript
// server/src/services/line.ts
import { WebClient } from '@line/bot-sdk';

const webClient = new WebClient(process.env.CHANNEL_ACCESS_TOKEN);

export const handleIncomingMessage = (event) => {
  // Send notification to user
  webClient.replyMessage(event.replyToken, {
    type: 'text',
    text: 'Hello, thanks for messaging us!',
  });
};

export const handleFollowEvent = (event) => {
  // Update user data
  console.log(`User ${event.source.userId} followed us`);
};
```
