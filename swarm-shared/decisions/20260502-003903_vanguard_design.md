# vanguard / design

### Diagnosis
* The Vanguard project lacks a comprehensive README file, making it difficult for new developers to understand the project's purpose, context, and functionality.
* The absence of a README file hinders the onboarding process for new team members and makes it challenging for them to contribute to the project.
* The project's directory structure and file organization are not well-documented, leading to potential confusion and difficulties in navigating the codebase.
* The lack of a README file also makes it harder for developers to understand the project's dependencies, requirements, and setup instructions.
* The project's recent commits and swarm-shared decisions suggest a need for improved documentation and knowledge sharing among team members.

### Proposed change
The proposed change involves creating a basic README file in the root directory of the project to provide an overview of the project, its purpose, and its functionality. The README file will be named `README.md` and will be located in the `/opt/axentx/vanguard` directory.
