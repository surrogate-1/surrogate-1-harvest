**HF CDN Bypass Implementation**
================================

**Task:** Implement the HF CDN Bypass to avoid Hugging Face API rate limits during data ingestion.

**Highest-Value Incremental Improvement:**

*   **HF CDN Bypass**: Public dataset files at `https://huggingface.co/datasets/{repo}/resolve/main/{path}` can be downloaded with NO Authorization header — bypasses /api/ auth-check rate limit entirely.

**Implementation Plan:**

### Step 1: Update `dataset-mirror` script to use HF CDN

Update the `dataset-mirror` script to download files from the HF CDN instead of using the Hugging Face API.

```bash
#!/bin/bash

# ...

# Download files from HF CDN
for file in $(curl -s -X GET \
  https://huggingface.co/datasets/${repo}/resolve/main/${path} \
  -H 'Accept: application/json' \
  -H 'Authorization: Bearer ${TOKEN}' \
  -H 'User-Agent: ${USER_AGENT}' \
  | jq -r '.files[] | @base64'); do
  # ...
done
```

### Step 2: Remove API calls from `dataset-mirror` script

Remove the API calls to list files and download files from the Hugging Face API.

```bash
#!/bin/bash

```

### Step 3: Update `train.py` to use HF CDN

Update the `train.py` script to use the HF CDN for data loading.
