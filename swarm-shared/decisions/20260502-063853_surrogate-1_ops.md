# surrogate-1 / ops

### **Diagnosis**
* The project lacks a robust implementation for handling Hugging Face API rate limits, which can block dataset training.
* The existing implementation may not be reusing existing Lightning Studio instances efficiently.
* The project does not have a mechanism to bypass the Hugging Face API rate limit for dataset training.
* The `granite-business-research.sh` script is not integrated with the knowledge-rag pipeline to provide contextual insights.
* The project does not have a clear strategy for handling Lightning Studio idle timeouts.

### **Proposed change**
The proposed change is to implement a mechanism to bypass the Hugging Face API rate limit for dataset training by using the HF CDN to download dataset files. This will involve modifying the `train.py` script to use the HF CDN URL to download dataset files instead of the Hugging Face API.

### **Implementation**
To implement this change, we will modify the `train.py` script as follows:
```python
import json
import os

# Load the list of dataset files from the JSON file
with open('dataset_files.json', 'r') as f:
    dataset_files = json.load(f)

# Define the HF CDN URL
hf_cdn_url = 'https://huggingface.co/datasets/{repo}/resolve/main/{path}'

# Download the dataset files from the HF CDN
for file in dataset_files:
    repo = file['repo']
    path = file['path']
    url = hf_cdn_url.format(repo=repo, path=path)
    os.system(f'wget {url}')
```
We will also add a script to pre-list the file paths once and embed them in the training script. This script will be run once after the rate-limit window clears.
```python
import requests

# Define the Hugging Face API URL
hf_api_url = 'https://huggingface.co/api/v1/datasets/{repo}/tree'

# Define the repository name
repo = 'axentx/surrogate-1'

# Get the list of dataset files from the Hugging Face API
response = requests.get(hf_api_url.format(repo=repo))
dataset_files = response.json()

# Save the list of dataset files to a JSON file
with open('dataset_files.json', 'w') as f:
    json.dump(dataset_files, f)
```
