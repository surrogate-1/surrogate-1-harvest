### High-Value Incremental Improvement: Implementing HF CDN Bypass for Faster Data Ingestion
#### Problem Statement
The current data ingestion pipeline in Airship is limited by Hugging Face API rate limits, causing delays in training and deployment of Surrogate AI models.

#### Solution Overview
To bypass the Hugging Face API rate limits, we will utilize the HF CDN to download dataset files directly. This approach eliminates the need for API calls during data loading, significantly reducing the ingestion time.

#### Implementation Plan

1. **Identify Dataset Repositories**: Determine the Hugging Face dataset repositories required for Surrogate AI model training.
2. **Pre-list File Paths**: Make a single API call to `list_repo_tree(path, recursive=False)` for each repository to retrieve a list of file paths.
3. **Save File Paths to JSON**: Store the retrieved file paths in a JSON file for later use.
4. **Modify Training Script**: Update the training script (`train.py`) to embed the JSON file containing the file paths.
5. **CDN-Only Fetches**: Configure the training script to download dataset files directly from the HF CDN using the embedded file paths.

#### Code Snippets

```python
import json
import requests

# Pre-list file paths for a repository
def pre_list_file_paths(repo_id, path):
    api_url = f"https://huggingface.co/api/v1/repo/{repo_id}/tree/{path}"
    response = requests.get(api_url)
    file_paths = response.json()["files"]
    return file_paths

# Save file paths to JSON
def save_file_paths_to_json(file_paths, json_file):
    with open(json_file, "w") as f:
        json.dump(file_paths, f)

# Example usage
repo_id = "dataset/repo"
path = "main"
file_paths = pre_list_file_paths(repo_id, path)
json_file = "file_paths.json"
save_file_paths_to_json(file_paths, json_file)
```

```python
import json
import requests

# Load file paths from JSON
def load_file_paths_from_json(json_file):
    with open(json_file, "r") as f:
        file_paths = json.load(f)
    return file_paths

# Download dataset files from HF CDN
def download_dataset_files(file_paths):
    for file_path in file_paths:
        cdn_url = f"https://huggingface.co/datasets/{file_path}/resolve/main/{file_path}"
        response = requests.get(cdn_url)
        # Save the file to the desired location
        with open(file_path, "wb") as f:
            f.write(response.content)

# Example usage
json_file = "file_paths.json"
file_paths = load_file_paths_from_json(json_file)
download_dataset_files(file_paths)
```

#### Deployment
Update the `docker-compose.microservices.yml` file to include the modified training script and JSON file containing the file paths.

```yml

services:

      - ./file_paths.json:/app/file_paths.json
```
