1. **Identify dataset files**: Pre-list file paths once using the `list_repo_tree` API call with `recursive=False` for one date folder.
2. **Save file list to JSON**: Embed the list of file paths in a JSON file.
3. **Modify training script**: Update the training script to use the CDN-only fetches with zero API calls during data load.
4. **Use CDN URL**: Use the `https://huggingface.co/datasets/{repo}/resolve/main/{path}` URL to download dataset files without authorization headers.
