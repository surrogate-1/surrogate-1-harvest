### Implementation Plan

1. **Identify dataset files**: Pre-list file paths once using the `list_repo_tree` API call with `recursive=False` for one date folder.
2. **Save file list to JSON**: Embed the list of file paths in a JSON file.
3. **Modify training script**: Update the training script to use the CDN-only fetches with zero API calls during data load.
4. **Use CDN URL**: Use the CDN URL `https://huggingface.co/datasets/{repo}/resolve/main/{path}` to download dataset files without authorization headers.

### Code Snippets

```python
import json
import requests

# Pre-list file paths once
def get_file_paths(repo, date_folder):
    url = f"https://huggingface.co/api/v1/datasets/{repo}/tree?path={date_folder}&recursive=False"
    response = requests.get(url)
    file_paths = [file["path"] for file in response.json()]
    return file_paths

# Save file list to JSON
def save_file_list(file_paths, json_file):
    with open(json_file, "w") as f:
        json.dump(file_paths, f)

# Modify training script to use CDN-only fetches
def download_dataset_file(file_path, repo):
    url = f"https://huggingface.co/datasets/{repo}/resolve/main/{file_path}"
    response = requests.get(url)
    return response.content

# Example usage
repo = "your-repo"
date_folder = "your-date-folder"
json_file = "file_paths.json"
file_paths = get_file_paths(repo, date_folder)
save_file_list(file_paths, json_file)

# In your training script
```
