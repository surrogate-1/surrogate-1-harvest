### Highest-Value Incremental Improvement for Airship Ops
#### Problem Statement
The airship project requires improvements in its ops, focusing on patterns and lessons learned from past experiences. The goal is to identify and implement the highest-value incremental improvement that can ship in <2h.

#### Solution
Based on the provided patterns and lessons learned, the highest-value incremental improvement for airship ops is to **implement a wrapper script for the market analysis script (e.g., granite-business-research.sh) to execute knowledge-rag and query top hub and related docs for contextual insights**.

#### Implementation Plan
1. Create a new wrapper script (e.g., `run_market_analysis.sh`) with the proper Bash shebang (`#!/usr/bin/env bash`) and make it executable (`chmod +x`).
2. In the wrapper script, invoke the market analysis script (e.g., `granite-business-research.sh`) and then execute knowledge-rag to query top hub and related docs.
3. Set `SHELL=/bin/bash` in the crontab to ensure the wrapper script is invoked via Bash.
4. Test the wrapper script to ensure it works as expected.

#### Code Snippet
```bash
#!/usr/bin/env bash

# Invoke market analysis script
./granite-business-research.sh

# Execute knowledge-rag to query top hub and related docs
knowledge-rag --query "top hub and related docs"
```
#### Example Use Case
To use the wrapper script, simply add it to the crontab and set the `SHELL` environment variable:
```bash
crontab -e
```
Add the following line:
```bash
SHELL=/bin/bash
0 0 * * * /path/to/run_market_analysis.sh
```
This will run the wrapper script daily at midnight, executing the market analysis script and querying top hub and related docs for contextual insights.
